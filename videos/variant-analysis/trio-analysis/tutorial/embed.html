Trio Analysis using Synthetic Datasets from RD-Connect GPAP

Video

Description:

To discover causal mutations of inherited diseases it’s common practice to do a trio analysis. In a trio analysis DNA is sequenced of both the patient and parents. Using this method, it’s possible to identify multiple inheritance patterns.
